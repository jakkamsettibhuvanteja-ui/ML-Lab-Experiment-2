# ML Lab Experiment 2 – Descriptive Statistics using Python

## 📌 Objective

To perform descriptive statistical analysis on two real-world datasets using Python and visualize the data distribution.

## 📂 Datasets Used

1. Titanic Dataset
2. Iris Dataset

## 🛠️ Technologies Used

- Python 3
- Pandas
- NumPy
- Matplotlib
- SciPy
- Jupyter Notebook
- VS Code

## 📊 Statistical Measures Performed

- Mean
- Median
- Mode
- Variance
- Standard Deviation
- Quartiles (Q1, Q2, Q3)
- Interquartile Range (IQR)
- Outlier Detection using IQR

## 📈 Data Visualizations

- Histogram
- Boxplot
